⚛️ Getting Started

GaussianHelper makes it simple to prepare Gaussian Job Files (.gjf) and configure calculation workflows. Follow these basic steps:

1. Load a Chemical Structure

Under the Molecule panel, choose your input method:

  • Draw Structure: Sketch structures manually inside the 2D canvas editor.
  • SMILES Input: Enter chemical strings (e.g. O for water, c1ccccc1 for benzene) to auto-generate 3D coords.
  • Upload File: Open chemical coordinate structures directly from .cdxml, .mol, .sdf, or .xyz files.

2. Configure Calculation Theory

In the Parameters panel, set parallel processor cores, memory parameters, checkpoint file path, method functional, basis sets, and implicit solvation.

💡 Pro Tip: Use the Presets dropdown in the top header to apply pre-configured energy optimization or frequency sweeps in a single click.

3. Preview & Validate Input File

Observe the live rendered 3D visualizer and `.gjf` syntax highlighter under the Current Job tab. Any parameter problems (like charge-multiplicity parity mismatches) will trigger warning alert bars.

4. Assemble Job Batches

Generate execution lists for multiple calculations under the Batch Queue tab:

  • Click Add to Batch inside the GJF code preview to queue the active configuration.
  • Upload existing .gjf files to batch-process historical setups.
  • Choose between Windows Batch Control (.bcf) or Linux Execution Command script (.sh) targets.
